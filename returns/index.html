Returns & Exchanges

Returns Accepted Exchanges Accepted
Return Window

Buyer must contact us and ship the item back within 30 days of delivery.

Conditions of Return

  • Buyer can return or exchange this item
  • Item must be returned within 30 days of delivery
  • Item must be in its original condition — unworn, unwashed, and with all original tags attached
  • Buyer is responsible for return shipping costs
  • Buyer is responsible for any loss in value if an item is not returned in original condition

How to Start a Return or Exchange

To initiate a return or exchange, please contact us through our Etsy shop or email us at jarod@mallorythreadtheory.com with:

  • Your order number
  • The item(s) you'd like to return or exchange
  • The reason for the return or exchange

We'll respond within 1–2 business days with return instructions.

Exceptions

The following items cannot be returned or exchanged:

  • Items that have been worn, washed, or damaged after delivery
  • Items returned more than 30 days after the delivery date
  • Custom or personalized orders
